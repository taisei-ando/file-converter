アプリ概要

ファイル変換くんは、画像の形式変換、ファイルサイズの軽量化、リサイズ、PDF作成、PDF画像化、動画処理、プライバシー情報削除などを行うユーティリティアプリです。

処理対象のファイルは原則として端末内で処理します。現行版では、当方が管理するサーバーへファイルをアップロードして変換する機能はありません。

ただし、広告表示、購入処理、問い合わせ、レビュー依頼、共有先アプリ、App StoreやGoogle Playなどの外部サービスでは、各サービスの仕様に従って情報が送信又は処理される場合があります。

広告と有料版

無料版では広告が表示される場合があります。有料版では、広告削除、テーマカラー変更など、購入画面に表示される機能を利用できます。

また、追加機能を伴わない任意の応援課金を提供する場合があります。購入、復元、返金等は各プラットフォームの定めに従います。
